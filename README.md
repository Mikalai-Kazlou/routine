### Users

Returns json data about users. Registration, authorization actions.

<details>

- `GET` /api/users
- `GET` /api/users/:id
- `POST` /api/registration
- `POST` /api/login
- `PUT` /api/users
- `DELETE` /api/users/:id

</details>

- `DELETE` /api/thoughts/:id

</details>
